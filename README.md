# Personal Portfolio — HTML/CSS Template

A clean, minimal single-page portfolio template built with pure HTML and CSS.
Feel free to use this structure as a starting point for your own portfolio.

## Structure

```
portfolio/
├── index.html        # Main HTML structure
├── style.css         # Your stylesheet (not included — add your own)
├── script.js         # JS for theme toggle + scroll reveal (not included)
└── images/           # Project screenshots and assets
```

## Sections

- **Hero** — Name, tagline, CTA buttons
- **About** — Bio + skills grid
- **Projects** — Project cards with hover overlay (supports GitHub links)
- **Services** — What you offer (description reveals on hover)
- **Contact** — Email, GitHub, LinkedIn links

## How to Use

1. Clone or download this repo
2. Edit `index.html` with your own content
3. Add your own `style.css` for custom styling
4. Drop your project images into the `images/` folder
5. Deploy on [Vercel](https://vercel.com) or GitHub Pages

## Deployment (Vercel CLI)

```bash
npm install -g vercel
cd your-portfolio-folder
vercel
```
